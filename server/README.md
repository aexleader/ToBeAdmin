## ThinkPHP Admin

Element 前端: https://github.com/xiaodit/think-element  

Ant Design 前端: https://github.com/xiaodit/think-ant-vue  


### 账号
* 超级管理员 `admin, 1234` 
* 普通管理员 `test, 1234`
